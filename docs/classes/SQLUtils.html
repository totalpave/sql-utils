StatictoValue

  • toValue(x: string | number | boolean | Date): string

  • Returns a "drop-in" string value of x. To be used to manually craft SQL queries without variable placeholder injections.

    NOT TO BE USED BY UNTRUSTED INPUT. This method does not protect against SQL injections.

    The conversions are as follows:

    • null/undefined becomes NULL
    • strings becomes double-quoted strings (e.g.) 'string' becomes '"string"'
    • Date gets formatted using toDatetime and then gets double-quoted
    • Boolean becomes 1 for true, and 0 for false
    • Number gets passed through as a regular string.

    Any other types is not supported and an error will be raised.

    Parameters

    • x: string | number | boolean | Date

    Returns string
